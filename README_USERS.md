# Stock Manager - Professional Edition

## 📦 Τι είναι;
Επαγγελματικό σύστημα διαχείρισης αποθέματος με GUI (Graphical User Interface).

## ✨ Χαρακτηριστικά
- ✅ Διαχείριση προϊόντων (προσθήκη, επεξεργασία, διαγραφή)
- ✅ Παρακολούθηση εισόδων/εξόδων
- ✅ Στατιστικά & Dashboard
- ✅ Εξαγωγή σε PDF & Excel
- ✅ Αυτόματη δημιουργία αντιγράφων ασφαλείας
- ✅ Αναζήτηση & φιλτράρισμα
- ✅ Υποστήριξη πολλαπλών κατηγοριών
- ✅ Ειδοποιήσεις χαμηλού αποθέματος

## 🚀 Εγκατάσταση (για χρήστες)

### Λήψη
1. Κατέβασε το αρχείο `StockManager.exe`
2. Κατέβασε τον φάκελο `data/` (περιέχει τα δεδομένα σου)

### Εκκίνηση
1. Βάλε το `StockManager.exe` και τον φάκελο `data/` στον ίδιο φάκελο
2. Κάνε διπλό κλικ στο `StockManager.exe`
3. Απόλαυσε!

**Δομή φακέλου:**
```
📁 StockManager/
├── StockManager.exe     ← Το πρόγραμμα
└── data/                ← Τα δεδομένα σου
    ├── products.json
    ├── movements.json
    ├── categories.json
    └── backups/
```

## 🔄 Ενημερώσεις (Updates)
Όταν κυκλοφορήσει νέα έκδοση:
1. Αντικατέστησε μόνο το `StockManager.exe`
2. **ΜΗΝ** διαγράψεις τον φάκελο `data/` (περιέχει τα προϊόντα σου!)

## 📊 Χρήση

### Βασικές Λειτουργίες
1. **Dashboard**: Στατιστικά σε πραγματικό χρόνο
2. **Προϊόντα**: Προσθήκη/Επεξεργασία προϊόντων
3. **Κινήσεις**: Καταγραφή εισόδων/εξόδων
4. **Εξαγωγές**: PDF & Excel reports
5. **Αντίγραφα Ασφαλείας**: Αυτόματα & χειροκίνητα

### Συντομεύσεις
- `Ctrl+S`: Αποθήκευση
- `Ctrl+F`: Αναζήτηση
- `ESC`: Ακύρωση
- `F5`: Ανανέωση

## 🔒 Ασφάλεια Δεδομένων
- ✅ Αυτόματη αποθήκευση σε κάθε αλλαγή
- ✅ Αυτόματα backups (20 τελευταία)
- ✅ Επιβεβαίωση πριν τη διαγραφή
- ✅ JSON format για εύκολη μεταφορά
