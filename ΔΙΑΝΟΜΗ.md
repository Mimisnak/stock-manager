# 🎉 Stock Manager - Οδηγίες Διανομής

## ✅ Το EXE έτοιμο!

### 📦 Τι να μοιράσεις:

Από τον φάκελο `stock_manager`, μοίρασε:

```
📁 StockManager/ (δημιούργησε νέο φάκελο)
├── 📄 StockManager.exe (από dist/)
├── 📁 data/
│   ├── products.json
│   ├── movements.json
│   ├── categories.json
│   └── 📁 backups/
└── 📄 README_USERS.md (οδηγίες χρήσης)
```

### 🔧 Βήματα:

1. **Δημιούργησε νέο φάκελο** `StockManager`
2. **Αντίγραψε** το `dist\StockManager.exe` εκεί

4. **Αντίγραψε** το `README_USERS.md` εκεί

## 🆘 Common Issues:

**"Windows Defender blocked it"**
- Φυσιολογικό για unsigned EXE
- Κάνε click "More info" → "Run anyway"
- Για επαγγελματική διανομή, χρειάζεται Code Signing Certificate

**"Missing data folder"**
- Βεβαιώσου ότι το `data/` είναι δίπλα στο EXE

**"Can't save changes"**
- Έλεγξε file permissions
- Τρέξε ως κανονικός χρήστης (όχι admin)
